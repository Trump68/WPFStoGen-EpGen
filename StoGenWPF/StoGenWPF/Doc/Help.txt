TRN: 
       [T.O.S.B.E]>[T.O.S.B.E]*[T.O.S.B.E]~
	T- type of transition
	O- Options value
	S- time of transition
	B- Begin value   (optional)
	E- End value     (optional)

		T: O - opacity
		   X- coord X transition
		   Y- coord Y transition
		   W- Wait (example W..3000)
		   S- scaleX (if relative - %)
		   C- scaleY (if relative - %)
		   A - scaleXY (if relative - %)
		   R - rotate (in degrees)

		o: A-триггер
		   B-линейная	
		   С-степенная (замедл)  Cn1n2e (n1 - отсечка остановки, если 1 - нет отсечки, в конце замедл до 0, n2 - степень [1-9], e- контроль по конечному значению (может не быть))

		   D-степенная (ускор)   Dn1n2e (n1 - отсечка остановки, если 1 - нет отсечки, n2 - степень [1-99], e- контроль по конечному значению (может не быть))
